# Крестики-нолики на PyQt5 / Tic-Tac-Toe on PyQt5

Десктопная **игра** «**крестики-нолики**» на **Python** с **GUI** на **PyQt5** — пример **Python game example** и небольшой **PyQt5 tutorial** для **QMainWindow**, **QPushButton** и **QLabel**. Подходит как **university lab work** (лабораторная работа).

## О проекте / About

Классическая **игра** **Tic-Tac-Toe** в виде **desktop app**: поле 3×3, ходы по очереди, проверка победы и ничьей. Проект демонстрирует базовую работу с **PyQt5** и виджетами окна.

## Возможности / Features

- Поле **3x3** для **игры** «**крестики-нолики**»
- Определение победы (строки, столбцы, диагонали)
- Определение ничьей
- Кнопка «Новая игра» без перезапуска приложения
- Отображение текущего хода игрока (**Player turn display**)

## Технологии / Technologies

- **Python** 3.x
- **PyQt5**
- **QMainWindow**, **QPushButton**, **QLabel**

## Установка / Installation

1. Клонируйте репозиторий: `git clone https://github.com/Saitama4722/lab5_tictactoe.git`
2. Перейдите в папку проекта: `cd lab5_tictactoe`
3. Установите **PyQt5**: `pip install PyQt5`
4. Запустите **игру**: `python main.py`

## Запуск / Usage

```bash
python main.py
```

# Tic-Tac-Toe on PyQt5 / Крестики-нолики на PyQt5

A **desktop app** **game** built with **Python** and **PyQt5** — a simple **GUI** **Tic-Tac-Toe** (**крестики-нолики**) demo that works as a **Python game example** and a short **PyQt5 tutorial** using **QMainWindow**, **QPushButton**, and **QLabel**. Useful as **university lab work**.

## About / О проекте

This project is a classic **Tic-Tac-Toe** **game** for the desktop: a 3×3 board, alternating moves, win and draw detection. It shows fundamental **PyQt5** window and widget usage.

## Features / Возможности

- **3x3** **game** board
- Win detection (rows, columns, diagonals)
- Draw detection
- New **game** button without restarting the app
- **Player turn display**

## Technologies / Технологии

- **Python** 3.x
- **PyQt5**
- **QMainWindow**, **QPushButton**, **QLabel**

## Installation / Установка

1. Clone the repository: `git clone https://github.com/Saitama4722/lab5_tictactoe.git`
2. Enter the project folder: `cd lab5_tictactoe`
3. Install **PyQt5**: `pip install PyQt5`
4. Run the **game**: `python main.py`

## Usage / Запуск

```bash
python main.py
```
